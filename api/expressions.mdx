## Overview

The Expression API provides functions for creating and manipulating Z3 abstract syntax trees (ASTs), which represent terms, formulas, and types.

## Types

### Z3_ast

Abstract syntax tree node - the fundamental data structure in Z3 for representing terms, formulas, and types.

### Z3_app

A kind of AST used to represent function applications. Subtype of Z3_ast.

### Z3_ast_kind

The different kinds of Z3 AST nodes:

<ParamField name="Z3_NUMERAL_AST" type="enum">
  Numeral constants
</ParamField>

<ParamField name="Z3_APP_AST" type="enum">
  Constants and applications
</ParamField>

<ParamField name="Z3_VAR_AST" type="enum">
  Bound variables
</ParamField>

<ParamField name="Z3_QUANTIFIER_AST" type="enum">
  Quantifiers (forall, exists)
</ParamField>

<ParamField name="Z3_SORT_AST" type="enum">
  Sort/type
</ParamField>

<ParamField name="Z3_FUNC_DECL_AST" type="enum">
  Function declaration
</ParamField>

### Z3_symbol

Lisp-like symbol used to name types, constants, and functions. Can be created using strings or integers.

## Symbol Creation

### Z3_mk_int_symbol

```c
Z3_symbol Z3_API Z3_mk_int_symbol(Z3_context c, int i);
```

Create a Z3 symbol using an integer.

<ParamField name="c" type="Z3_context">
  Context
</ParamField>

<ParamField name="i" type="int">
  Integer value (must be in range 0 to 2^30-1)
</ParamField>

<ResponseField name="return" type="Z3_symbol">
  New symbol
</ResponseField>

### Z3_mk_string_symbol

```c
Z3_symbol Z3_API Z3_mk_string_symbol(Z3_context c, Z3_string s);
```

Create a Z3 symbol using a C string.

<ParamField name="c" type="Z3_context">
  Context
</ParamField>

<ParamField name="s" type="Z3_string">
  String name
</ParamField>

<ResponseField name="return" type="Z3_symbol">
  New symbol
</ResponseField>

**Example:**

```c

Z3_symbol y_sym = Z3_mk_string_symbol(ctx, "y");
```

## Complete Example

```c
#include <z3.h>

int main() {
    Z3_config cfg = Z3_mk_config();
    Z3_context ctx = Z3_mk_context(cfg);
    Z3_del_config(cfg);
    
    // Create integer sort
    Z3_sort int_sort = Z3_mk_int_sort(ctx);
    
    // Create variables x and y
    Z3_ast x = Z3_mk_const(ctx, Z3_mk_string_symbol(ctx, "x"), int_sort);
    Z3_ast y = Z3_mk_const(ctx, Z3_mk_string_symbol(ctx, "y"), int_sort);
    
    // Create expression: x + y = 10
    Z3_ast args[2] = {x, y};
    Z3_ast sum = Z3_mk_add(ctx, 2, args);
    Z3_ast ten = Z3_mk_int(ctx, 10, int_sort);
    Z3_ast constraint = Z3_mk_eq(ctx, sum, ten);
    
    Z3_del_context(ctx);
    return 0;
}
```
